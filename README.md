# Stock Time Series Analysis using Python

## Project Overview
This project performs Time Series Analysis on Apple stock price data using Python, Pandas, and Matplotlib.

The project focuses on analyzing stock market trends, visualizing stock price movements, and applying moving average smoothing techniques to identify long-term trends.

---

## Tools & Technologies Used
- Python
- Pandas
- Matplotlib
- Google Colab
- GitHub

---

## Dataset Information
The dataset contains historical stock market data with the following columns:

- Symbol
- Date
- Open Price
- High Price
- Low Price
- Close Price
- Volume

For this analysis, Apple (AAPL) stock data was used.

---

## Steps Performed
1. Data Loading using Pandas
2. Date Conversion & Time Indexing
3. Data Filtering for Apple Stock
4. Stock Price Trend Visualization
5. 30-Day Moving Average Calculation
6. Time Series Visualization
7. Trend Interpretation & Insight Generation

---

## Visualizations Created
- Apple Stock Closing Price Trend
- 30-Day Moving Average Trend

---

## Key Insights
- Apple stock prices showed a strong upward trend between 2014 and 2018.
- The stock experienced regular short-term market volatility.
- The 30-day moving average successfully smoothed daily price fluctuations.
- Moving averages helped identify long-term market direction and trend behavior.
